# 伪元素

## 什么是伪元素

## 怎样使用伪元素

### 常见的用法

```css
div:hover {
}
input:focus {
}
/* input type为单选，被选中时 */
input:checked {
  color: #333;
}
/* input 被选中时 它的兄弟元素 span 的样式 */
input:checked ~ span {
  color: #333;
}

/* input 被禁用时 */

input:disabled {
  color: #333;
}
```
